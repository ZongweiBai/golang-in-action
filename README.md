# golang-in-action

Golang's entry project integrates common components and functions such as gin, swagger, JWT, vaper, zap, redis, etc.

# How to use

```bash
# install golang mod
go list
go mod tidy
```

# Generate Docs

```bash
$GOPATH/bin/swag init
```

And then access url: http://localhost:8080/api-docs/swagger/index.html
